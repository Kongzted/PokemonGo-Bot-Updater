Welcome to PokemonGo Bot Updater!

  • Download the script from GitHub here.
  • Create a folder where the script and bot can be placed in.
  • Extract the content of the 'script.rar' file to the folder you just created.
  • Run 'Update-Bot.sh' script from Git Bash Terminal.
  • Go to 'PokemonGo-Bot/Builds Only' and extract the .rar archive in that folder.
  • Run the 'Pokemon.Go.RocketAPI.Console.exe'.
  • Fill out the necessary informations and thats it.
